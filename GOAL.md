# Goal (MVP)

Build a web platform for amateur football players to watch, relive, and share their match highlights shortly after the game ends.

The platform should focus on video consumption, player identity, and easy sharing.

---

## Core Features

- [ ] Tournament list view (browse available tournaments)
- [ ] Match list by date (inside each tournament)
- [ ] Match detail page with:
  - [ ] Full match video
  - [ ] Highlight clips (20–30 seconds each)
- [ ] Highlight card with:
  - [ ] Player name
  - [ ] Type of play (Goal, Save, Assist, etc.)
  - [ ] Timestamp
  - [ ] Share button
- [ ] Player profile page:
  - [ ] Profile photo
  - [ ] Position
  - [ ] List of personal highlights
- [ ] Simple public highlights feed (latest or top plays)
- [ ] Data stored locally (mock data or localStorage)

---

## Categories (Play Types)

Use these default highlight types:

- Goal
- Assist
- Save
- Shot on Target
- Defensive Play
- Other

---

## User Roles (Basic)

- Player (can view and share highlights)
- Viewer (can browse tournaments and watch highlights)

---

## Design Direction

- Clean, modern sports interface
- Dark mode by default
- Video-first layout (large video area)
- Mobile-friendly layout
- Card-based highlight grid
- Simple typography (no custom fonts)
- Minimal dashboard UI (focus on content, not admin)

---

## Future Improvements (Optional)

- Auto-generated player highlight reel
- Download vertical version for social media
- Simple stats per player
- Commentator audio overlay
- Reactions (likes or emojis)
